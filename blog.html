What is Document Object Model (DOM)?

DOM is a method to divide an HTML document into nodes and objects in order to manipulate them using a scripting language such as JavaScript.

How do you select an element from DOM?

An element can be selected from DOM by the ID or Class or even the general Tag of the element.

What is event delegation in the context of the DOM, and why is it useful?

Attaching an event listener to a parent node in order to handle events triggered by any of its child elements is called event delegation. It increases performance significantly and simplifies code.

How do you manipulate an element's attributes and styles using DOM?

Using the element selection menthod to select the element we want to modify, wew can use the setAttribute property to change attributes and the style property to change styles.
